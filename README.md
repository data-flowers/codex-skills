# data.flowers Codex Skills

This repo publishes installable Codex skills for data.flowers workflows.

## Available skills

- `ptah-data-flow`

## Install

Use Codex's GitHub skill installer against this repo:

```bash
python /Users/hengxuli/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo data-flowers/codex-skills \
  --path skills/ptah-data-flow
```

Restart Codex after installing a new skill.
